# ahtml — Python consumer SDK for AHTML

Consume [AHTML](https://github.com/DibbayajyotiRoy/AHTML) — the agent-facing
web snapshot format — from Python: fetch with content negotiation and
ETag/TTL caching, parse both serializations, verify detached-JWS signatures
via `did:web`, and execute actions behind the same safety gate the
TypeScript `@ahtmljs/agent` uses.

```bash
pip install ahtml            # httpx + cryptography only
pip install 'ahtml[tokens]'      # + tiktoken token counting
pip install 'ahtml[langchain]'   # + LangChain document loader
```

Consumer-only by design (ADR-0001): producers are JS-framework-bound;
agents are not.

## Quickstart — answer a price question with LangChain

```python
from ahtml import AHTMLClient, run_action
from ahtml.langchain import AHTMLLoader

docs = AHTMLLoader("https://ahtmljs.com").load()          # snapshot → Document[]
price_facts = [d for d in docs if "price" in d.page_content]
print(price_facts[0].page_content)                         # grounded answer, no scraping

client = AHTMLClient()
snap = client.fetch("https://ahtmljs.com")                # ETag/TTL-cached
result = run_action(snap, "subscribe", {"tier": "pro"},
                    bearer="tok", dry_run=True)            # rehearse before paying
print(result.would_charge, result.would_side_effects)
```

## Safety gate

`run_action` refuses — raising `ActionRefused`, never touching the wire — when
the action contract demands more than the call provides:

- `confirmation: "required"` without `confirm=True`
- `auth: "required"` without a `bearer`
- site policy `agents_welcome: false`

`dry_run=True` returns the predicted charge and side effects without
executing. Policy presets (`POLICY_PRESETS`) mirror `@ahtmljs/agent` 1:1.

## Conformance

The test suite round-trips fixtures generated by the TypeScript reference
implementation **byte-identically** (canonical JSON, SPEC.md §1.1) and
verifies signatures produced by `@ahtmljs/schema`'s `signSnapshot`. See
`tests/` and the AHTML conformance corpus.
